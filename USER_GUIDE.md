# 🧠 What Is VAXINX Doppio?

VAXINX Doppio is a lightweight dashboard prototype exploring how wallet-connected identity systems may simplify notifications, app access, and modular digital workflows.

This project combines:
- notification hub concepts
- wallet-connect UX
- cyberpunk dashboard design
- GitHub Pages deployment
- lightweight static architecture

The current version is:
- experimental
- educational
- front-end only
- prototype stage (`v1`)

---

# ⚠️ Important Wallet Safety Notice

VAXINX Doppio:
- NEVER asks for your wallet recovery phrase
- NEVER stores seed phrases
- NEVER stores private keys
- DOES NOT process real authentication

Your wallet recovery phrase must remain:
- private
- offline
- secure
- never shared with any website or application

This project only explores wallet-connect style UX concepts.
