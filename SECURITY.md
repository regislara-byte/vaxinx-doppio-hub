# ⚠️ Prototype Status

VAXINX Doppio is an experimental front-end prototype.

This repository:
- is NOT a production security platform
- is NOT a real authentication system
- is NOT audited
- is NOT intended for financial use
- is NOT intended for sensitive account management

---

# 🔐 Wallet Safety

VAXINX Doppio:
- does NOT collect seed phrases
- does NOT store private keys
- does NOT manage wallets
- does NOT handle transactions

Wallet integrations are limited to:
- MetaMask-style account request concepts
- front-end wallet detection
- UI experimentation

---

# 🚫 Never Share

Never share:
- seed phrases
- wallet recovery phrases
- private keys
- backup codes
- passwords

with:
- websites
- applications
- strangers
- Discord users
- social media accounts

No legitimate wallet application should request your recovery phrase inside a dashboard UI.

---

# 🦊 MetaMask Usage

If using MetaMask:
- install only from official sources
- verify browser extensions carefully
- avoid phishing websites
- review wallet permissions regularly

Official MetaMask website:

https://metamask.io

---

# 🌐 Static Prototype Design

This repository is intentionally lightweight.

Current architecture:
- static HTML
- CSS
- JavaScript
- GitHub Pages deployment
- no backend database
- no server-side authentication

This reduces:
- infrastructure complexity
- hosting cost
- backend attack surface
