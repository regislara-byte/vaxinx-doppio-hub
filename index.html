Core Features
🔑
Wallet-Based Identity

Your Web3 wallet is your login. No passwords, no email sign-ups — just connect and your identity follows you across every module.

🔔
Unified Notifications

Aggregate signals from Discord, Telegram, X and on-chain events into a single feed. Never miss a drop, announcement, or community alert again.

⚡
Modular App Store

Browse and plug in notification modules on demand. Add YouTube, Instagram, WhatsApp and more without rebuilding your entire setup.

🛡️
Non-Custodial by Design

Zero custody. Zero seed phrase requests. The dashboard never touches your keys — wallet interaction is handled entirely through MetaMask's own API.

🌐
Static Deployment

Runs on GitHub Pages with zero server costs. Fully open source and deployable to any static host in minutes — Netlify, Vercel, Cloudflare Pages.

🎛️
Cyberpunk Dashboard UI

A distinctive dark-mode command centre aesthetic built for the Web3 generation. Designed to be memorable, not generic.
